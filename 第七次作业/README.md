# [第七次作业：创意编程研究：复杂性｜自相似与分形]作业：自相似与分形

- 结合课程案例，创建分形机制，建立一个具有生命特征/生态系统特征的可交互的动态创意作品，并输出为mp4/gif文件。

## 项目内容

语言：Processing

实现：通过对圆的分形，设置绘制圆时候的RGB和透明图，实现如图的艺术效果，单击鼠标绘制一幅分形图，当足够多的分形图叠加在一起时，形成渐变的艺术效果。

暗示了自然界中聚落的现象，蕴含了积少成多，量变引起质变的哲学思想。
