# [第五次作业：创意编程元素：图像/交互]作业

- 至少有两个功能按钮，这些按钮的具体功能（如画笔样式、颜色、动作等）请自行设定；

## 项目内容

语言：Processing

实现：利用if-else逻辑和基本的绘图函数实现简单的绘图工具，功能包括：背景色选择、线条粗细选择、线条颜色选择、清除画布
